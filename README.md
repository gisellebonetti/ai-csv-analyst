# 📊 AI CSV Analyst

Um aplicativo de análise de dados que transforma arquivos CSV em insights automáticos usando Python, visualização de dados e Inteligência Artificial local.

---

## 🚀 Visão Geral

O **AI CSV Analyst** é uma ferramenta interativa que permite analisar dados de forma rápida e inteligente.

O usuário pode enviar um arquivo CSV e o sistema automaticamente:

- Processa os dados
- Gera gráficos interativos
- Calcula estatísticas
- Gera insights com IA local

---

## 🧠 Problema que resolve

Análises manuais de dados em planilhas podem ser lentas e pouco intuitivas.

Este projeto resolve isso ao transformar dados brutos em:

- Visualizações claras
- Resumos automáticos
- Insights gerados por IA

---

## ⚙️ Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- Ollama (IA local)

---

## 🏗️ Arquitetura do sistema
CSV → Pandas → Streamlit → Ollama → Insights

---

## 📊 Funcionalidades

- Upload de arquivos CSV
- Visualização de dados em tabela
- Gráficos automáticos (linha e barras)
- Estatísticas descritivas
- Geração de insights com IA local

---

## 🧠 Inteligência Artificial

O projeto utiliza um modelo de linguagem local rodando via Ollama para:

- Analisar os dados
- Identificar padrões
- Gerar explicações em linguagem natural

Isso permite análise sem necessidade de APIs externas.
